# is-eight
NPM package for checking if a number is equal to 8.

## Installation

``` bash
$ npm install is-eight
```

## Usage

``` js
var eight = require('is-eight');

if(eight(8))
{
  console.log("It is eight indeed!");
}
else
{
  console.log("It is not eight. What a shame!");
}
```
