# is-not-eight
NPM package for checking if a number is not equal to 8.

## Installation

``` bash
$ npm install is-not-eight
```

## Usage

``` js
var notEight = require('is-not-eight');

if(notEight(8))
{
  console.log("It is eight indeed!");
}
else
{
  console.log("It is not eight. What a shame!");
}
```
